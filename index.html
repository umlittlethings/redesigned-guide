🩺 Framingham Risk Score API

REST API untuk menghitung risiko penyakit jantung berdasarkan data medis pengguna.

🔗 Endpoint

POST /api/framingham

📥 Request Body

{
  "user": "John Doe",
  "age": 45,
  "gender": "male",
  "totalCholesterol": 210,
  "hdlCholesterol": 50,
  "systolicBP": 130,
  "isSmoker": true,
  "isDiabetic": false
}

📤 Response

{
  "user": "John Doe",
  "riskScore": 18.54,
  "riskLevel": "medium",
  "message": "Pasien memiliki risiko medium terkena penyakit jantung."
}

📊 Risk Level

Skor Kategori Risiko
< 10 Low
10 - 19.9 Medium
≥ 20 High

📦 Tech Stack

  • Node.js
  • Express.js
  • dotenv
  • CORS
  • PostgreSQL
  • Sequelize
