🩺 Framingham Risk Score API

REST API untuk menghitung risiko penyakit jantung berdasarkan data medis pengguna.

🔗 Endpoint

POST /api/framingham

📥 Request Body

{
  "user": "John Doe",
  "age": 45,
  "gender": "male",
  "totalCholesterol": 210,
  "hdlCholesterol": 50,
  "systolicBP": 130,
  "isSmoker": true,
  "isDiabetic": false,
  "restingHeartRates": [78, 80, 82]
}

📤 Response

{
  "user": "John Doe",
  "riskScore": 18,
  "riskLevel": "medium",
  "riskPercentage": "5-15%",
  "avgHeartRate": 80,
  "message": "Pasien memiliki risiko medium (5-15%) terkena penyakit jantung dalam 10 tahun ke depan."
}

📊 Risk Level

Skor Kategori Risiko Estimasi Risiko 10 Tahun
≤ 9 Low ≤ 5%
10 - 19 Medium 5-15%
≥ 20 High > 15%

📦 Tech Stack

  • Node.js
  • Express.js
  • dotenv
  • CORS
  • PostgreSQL
  • Sequelize
